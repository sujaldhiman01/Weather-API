# Weather API 🌦️

A simple and responsive weather application that fetches real-time weather data using a weather API. Users can search for any city to view current weather conditions such as temperature, humidity, and wind speed through a clean and interactive interface.

## 🚀 Features
- Search weather by city name
- Real-time weather data
- Displays temperature, humidity, and wind information
- Dynamic weather icons based on conditions
- Simple and responsive user interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## ⚙️ How It Works
The application sends a request to the weather API based on the user’s input city name. The API returns weather data in JSON format, which is then processed and displayed on the webpage.
